php Object Oriented Programming

Contoh webiste yang
di develop menggunakan php OOP
01
Facebook

Facebook menggunakan PHP, terutama menggunakan HipHop Virtual Machine (HHVM) dan Hack, yang merupakan ekstensi PHP yang dikembangkan oleh Facebook. Sebagian besar kode Facebook menggunakan prinsip OOP, tetapi mungkin juga memanfaatkan konsep lain untuk meningkatkan performa dan skalabilitas.
02
Wikipedia

Wikipedia yang berjalan di MediaWiki juga menggunakan PHP. MediaWiki memiliki sejumlah aspek OOP, terutama dalam pembuatan ekstensi dan pengembangan modul, meskipun beberapa bagian mungkin masih menggunakan pendekatan prosedural.
03
WordPress

WordPress menggunakan PHP dan telah meningkatkan penerapan OOP-nya seiring waktu. Meskipun WordPress memiliki beberapa bagian yang masih bersifat prosedural, versi terbaru dan banyak plugin dan tema dikembangkan dengan pendekatan OOP.
04
Joomla

Joomla sebagai sistem manajemen konten (CMS) yang ditulis menggunakan PHP, memanfaatkan berbagai konsep dan prinsip pemrograman berorientasi objek (OOP). Beberapa teknologi PHP OOP yang digunakan dalam pengembangan Joomla melibatkan penggunaan kelas, objek, pewarisan, polimorfisme, dan konsep-konsep OOP lainnya.
